# Day 1 - Hello, World! Printer

> **One Day One Py Script** · Phase 1 : Python Foundations

## Code

```python
print("Hello, World")
```

## Output

```
Hello, World
```

## Concept

| Concept | อธิบาย |
| --- | --- |
| `print()` | สำหรับการแสดงผลข้อความออกมาทางหน้าจอ  |
| `"..."` | ข้อความอยู่ในเครื่องหมายคำพูด |

## สิ่งที่ค้นพบ 

- Python ไม่ต้องมี `main()` แบบภาษาอื่นๆ ก็สามารถรันได้เลย
- `"Hello"` และ `'Hello'` ใช้ได้เหมือนกัน ผลลัพธ์ออกมาเหมือนกัน
- `print()` ขึ้นบรรทัดใหม่ให้อัตโนมัติทุกครั้ง โดยที่เราไม่ต้อง `\n` เหมือนภาษาอื่นๆในการขึ้นบรรทัดใหม่
